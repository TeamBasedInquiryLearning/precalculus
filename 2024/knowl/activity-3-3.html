Activity 4.3.3.

Repeat this process, using long division, given \(f(x)=\frac{6x^2+5x-10}{2x+3}\text{.}\)
(a)
What is the result when \(6x^2\) is divided by \(2x\text{?}\) Place this number at the top left when doing long division.
  1. \(\displaystyle 3\)
  2. \(\displaystyle 3x\)
  3. \(\displaystyle 3x^2\)
(b)
Next, multiply this result by \(2x+3\) and write this below \(6x^2+5x-10 \text{.}\) Be sure to line up like terms in the same column. What is the result?
  1. \(\displaystyle 6x+9\)
  2. \(\displaystyle 6x^2+9x\)
  3. \(\displaystyle 6x^3+9x^2\)
(c)
Now subtract the like terms by placing them in the corresponding columns.
  1. \(\displaystyle 6x^2-x-19\)
  2. \(\displaystyle -4x-10\)
  3. \(\displaystyle -6x^3+5x-10\)
  4. \(\displaystyle 14x-10\)
(d)
Next, divide the first term in the previous result by \(2x\text{.}\) Write the answer at the top beside the first answer. What is the expression at the top now?
  1. \(\displaystyle 3-3x^2\)
  2. \(\displaystyle 3x-2\)
  3. \(\displaystyle 3x^2-3x^2\)
  4. \(\displaystyle 3x+7\)
(e)
Multiply this resulting term by \(2x+3\) and write that answer in the proper columns of like terms. What is this result?
  1. \(\displaystyle -6x^3-9x^2\)
  2. \(\displaystyle -4x-6\)
  3. \(\displaystyle 14x+21\)
(f)
Subtract the like terms. This is the remainder. What is the remainder?
  1. \(\displaystyle -6x^3-9x^2-4x-10\)
  2. \(\displaystyle -4\)
  3. \(\displaystyle -18x-31\)
(g)
What is the quotient?
  1. \(\displaystyle 3-3x^2\)
  2. \(\displaystyle 3x-2\)
  3. \(\displaystyle 0\)
  4. \(\displaystyle 3x+7\)
(h)
How can you check your answer? (Hint: Think of regular long division with positive integers.)
