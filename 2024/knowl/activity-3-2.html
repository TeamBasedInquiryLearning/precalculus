(a)
Simplify the following rational expression using factoring. \(f(x)=\frac{x^2+2x-8}{x+4}\)

  2. \(\displaystyle x-4\)
  3. \(\displaystyle x-2\)
  4. \(\displaystyle x+2\)
Now use long division to find the quotient and remainder.
(b)
What is the quotient?
  1. \(\displaystyle x+4\)
  2. \(\displaystyle x-4\)
  3. \(\displaystyle x-2\)
  4. \(\displaystyle x+2\)
(c)
What is the remainder?
  1. \(\displaystyle x+4\)
  2. \(\displaystyle x-4\)
  3. \(\displaystyle x-2\)
  4. \(\displaystyle 0\)
