Multiply the numerator and denominator of \(r\) by \(\frac{1}{x^2}\text{.}\) Then distribute and simplify as much as possible in both the numerator and denominator to write \(r\) in a different algebraic form. Which of the following is that new form?
  1. \(\displaystyle \frac{3x^4 - 5x^3 +x^2}{7x^4+2x^3-11x^2}\)
  2. \(\displaystyle \frac{3 - \frac{5}{x} + \frac{1}{x^2}}{7 + \frac{2}{x} - \frac{11}{x^2}}\)
  3. \(\displaystyle \frac{\frac{3x^2}{x^2} - \frac{5x}{x^2} + \frac{1}{x^2}}{\frac{7x^2}{x^2} + \frac{2x}{x^2} - \frac{11}{x^2}}\)
  4. \(\displaystyle \frac{3x^2 - 5x +1}{7x^4+2x^3-11x^2}\)
