Observation 4.6.4.

Let \(p\) and \(q\) be polynomial functions so that \(r(x) = \frac{p(x)}{q(x)}\) is a rational function. The domain of \(r\) is the set of all real numbers except those for which \(q(x) = 0\text{.}\)
