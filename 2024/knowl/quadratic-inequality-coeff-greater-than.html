Activity 1.7.1.

Recall that when solving a quadratic equation, you can first try to factor the quadratic. Solving inequalities will require multiple steps. Solve the quadratic inequality
\begin{equation*} x^2-4x \gt 32\text{.} \end{equation*}
(a)
First, move all terms to one side. Then factor the quadratic function,
\begin{equation*} f(x)=x^2-4x-32\text{.} \end{equation*}
  1. \(\displaystyle (x-4)(x+8)\)
  2. \(\displaystyle (x-8)(x+4)\)
  3. \(\displaystyle (x-8)(x-4)\)
  4. \(\displaystyle (x+8)(x+4)\)
(b)
Now set each factor \(=0\) and solve for \(x\text{.}\) These are the \(x\) intercepts.
  1. \(-8\text{,}\) \(-4\)
  2. \(8\text{,}\) \(4\)
  3. \(-8\text{,}\) \(4\)
  4. \(8\text{,}\) \(-4\)
(c)
Create a number line and place the values from each factor on this number line. We will now test all \(3\) intervals between the \(2\) values to see if we get a positive or a negative result. Record these signs in the intervals and shade the desired result. This is called a sign chart. We want only the intervals that are \(\gt 0\text{,}\) i.e. positive.
  1. \(x \gt 8\text{,}\) \(x \lt -4\)
  2. \(x \gt -8\text{,}\) \(x \gt -4\)
  3. \(x \lt 8\text{,}\) \(x \lt 4\)
  4. \(x \lt -8\text{,}\) \(x \gt 4\)
(d)
Write the intervals using interval notation.
  1. \(\displaystyle (-8,4)\)
  2. \(\displaystyle ( 4,8)\)
  3. \(\displaystyle (-\infty,-4)\cup(8,\infty)\)
  4. \(\displaystyle (-\infty,-8)\cup(4,\infty)\)
