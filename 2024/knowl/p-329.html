Create a number line and place the values from each factor on this number line. We will now test all \(3\) intervals between the \(2\) values to see if we get a positive or a negative result. Record these signs in the intervals and shade the desired result. This is called a sign chart. We want only the intervals that are \(\gt 0\text{,}\) i.e. positive.
  1. \(x \gt 8\text{,}\) \(x \lt -4\)
  2. \(x \gt -8\text{,}\) \(x \gt -4\)
  3. \(x \lt 8\text{,}\) \(x \lt 4\)
  4. \(x \lt -8\text{,}\) \(x \gt 4\)
