Activity 1.7.4.

Solve the rational inequality
\begin{equation*} \frac{5x-3}{x+4} \gt 7\text{.} \end{equation*}
(a)
First, move all terms to one side. Then find the common denominator and combine to one simplified fraction.
  1. \(\displaystyle \frac{12x+31}{x+4}\)
  2. \(\displaystyle \frac{12x-31}{x+4}\)
  3. \(\displaystyle \frac{-2x+31}{x+4}\)
  4. \(\displaystyle \frac{-2x-31}{x+4}\)
(b)
Now set both the numerator and denominator \(=0\) and solve for \(x\text{.}\) These are the partition values to place on the number line for testing intervals.
  1. \(\displaystyle -\frac{31}{2},-4\)
  2. \(\displaystyle \frac{31}{2},-4\)
  3. \(\displaystyle \frac{31}{12},-4\)
  4. \(\displaystyle -\frac{31}{12},-4\)
(c)
Create a number line and place the values from each factor on this number line. We will now test all intervals between the \(2\) values to see if we get a positive or a negative result. Record these signs in the intervals and shade the desired result. This is called a sign chart. We want only the intervals that are \(\gt 0\text{,}\) i.e. positive.
  1. \(x \gt -\frac{31}{2}\text{,}\)\(x \lt -4\)
  2. \(x \gt \frac{31}{2}\text{,}\)\(x \lt -4\)
  3. \(x \gt -\frac{31}{2}\text{,}\)\(x \lt 4\)
  4. \(x \lt \frac{31}{2}\text{,}\)\(x \gt 4\)
(d)
Write the intervals using interval notation.
  1. \(\displaystyle \left(-\frac{31}{2},4\right)\)
  2. \(\displaystyle (4,8)\)
  3. \(\displaystyle (-\infty,-4)\cup(8,\infty)\)
  4. \(\displaystyle (-\infty,-8)\cup(4,\infty)\)
