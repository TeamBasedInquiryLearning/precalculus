Activity 1.1.3.

The following linear equations appear similar, but their solutions are very different.

(a)

Which of these equations has one unique solution?
  1. \(\displaystyle 4(x-2)=4x+6\)
  2. \(\displaystyle 4(x-1)=4x+4\)
  3. \(\displaystyle 4(x-1)=x+4\)

(b)

Which of these equations has many solutions?
  1. \(\displaystyle 4(x-2)=4x+6\)
  2. \(\displaystyle 4(x-1)=4x+4\)
  3. \(\displaystyle 4(x-1)=x+4\)

(c)

Which of these equations has no solutions?
  1. \(\displaystyle 4(x-2)=4x+6\)
  2. \(\displaystyle 4(x-1)=4x+4\)
  3. \(\displaystyle 4(x-1)=x+4\)
